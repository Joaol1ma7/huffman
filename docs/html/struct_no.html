Detailed Description

Estrutura de nó para árvores de Huffman e listas encadeadas.

Esta estrutura é utilizada para representar nós em uma árvore de Huffman , armazenando um caractere , sua frequência e ponteiros para os nós esquerdo, direito e próximo(Para listas encadeadas).
